There are 4 ways to declare a javascript variable:
  • Using var
  • Using let
  • Using const
  • Using nothing
Variables are containers for storing data(storing data values). In this example, logging to console, variables are declared by the let: 
			{
				<script>let x=5;let y=6;let z=x+y;
				console.log(z)
		}
Note that as we use let and below we again declare the same variables with var keyword, we wrote the above code in {} block, not to get an error

In this example, x,y,and z, are variables declared with the var keyword: 

			<script>var x=5;var y=6;var z=x+y;
			alert(z)
			<script>
