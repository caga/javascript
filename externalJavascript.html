Placing scripts in external files has sore advantages:
  • It separetes HTML and code
  • It makes HTML and JavaScript easier to read and maintain
  • Cached JavaScript files can speed up page loads
To add several script files to one page -use several script tags: 
			<script src="static/scripts/myScript.js"></script>
			<script src="static/scripts/myScript2.js"></script>

External Referances

An external script can be referenced in F different ways:
  • With afull URL (a full web address)
  • With a file path (like /js/
  • Without any path
